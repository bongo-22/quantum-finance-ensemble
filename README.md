## О проекте

Проект исследует работоспособность гибридного ансамбля, состоящего из нескольких вариационных квантовых классификаторов (Variational Quantum Classifier — VQC) с различными feature maps, чьи выходы агрегируются классической мета-моделью (stacking), для задачи бинарной классификации направления движения цены акции Apple Inc. (тикер AAPL) за период 2020–2024.

**Основная цель** — определить:
- превосходит ли ансамбль одиночные квантовые модели;
- способен ли он конкурировать с классическими моделями (Logistic Regression, RandomForest, XGBoost);
- возникает ли прирост качества за счёт разнообразия квантовых представлений данных.

## Цели

1. Сравнить одиночные VQC с классическими моделями машинного обучения.
2. Исследовать влияние архитектуры квантовой схемы (feature map, глубина reps) на качество классификации и время обучения.
3. Проверить устойчивость VQC к случайной инициализации (seed-анализ).
4. Реализовать stacking ансамбль из RandomForest и лучшей VQC модели.
5. Проанализировать зависимость метрик ROC-AUC от архитектуры и сформулировать математическое обоснование наблюдаемых результатов.

## Постановка задачи

**Бинарная классификация направления доходности:**

$$r_t = \frac{P_t - P_{t-1}}{P_{t-1}}$$

$$y_t = \begin{cases} 1, & r_{t+1} > 0 \\ 0, & r_{t+1} \leq 0 \end{cases}$$

**Признаки:**
- Return — дневная доходность
- MA5 — скользящее среднее за 5 дней
- MA10 — скользящее среднее за 10 дней
- Volatility — волатильность (стандартное отклонение доходности за 5 дней)

**Метрики:**

$$Accuracy = \frac{TP + TN}{TP + TN + FP + FN}$$

$$AUC = \int_0^1 TPR(FPR) \, d(FPR), \quad TPR = \frac{TP}{TP+FN}, \quad FPR = \frac{FP}{FP+TN}$$

**Интерпретация ROC-AUC:**
$$AUC = P(s(x^+) > s(x^-))$$
вероятность того, что случайный положительный объект получит больший прогнозный скор, чем случайный отрицательный.

## Вариационные квантовые классификаторы (VQC)

Variational Quantum Classifier (VQC) — это квантово-классическая модель, в которой классификация осуществляется с помощью параметризованной квантовой схемы.

**Математически:**

1. **Кодирование данных (Feature Map)**: $x \mapsto |\psi(x)\rangle = U_{\phi(x)}|0\rangle$
2. **Вариационная схема (Ansatz)**: $|\psi(\theta, x)\rangle = U(\theta)|\psi(x)\rangle$
3. **Измерение**: $\phi(x) = \langle \psi(\theta, x)| O |\psi(\theta, x)\rangle$
4. **Классификация**: $p(y=1|x) = \sigma(\phi(x))$

**Физический смысл:**
- Вращения = настраиваемые фильтры для признаков
- Запутывание = нелинейное взаимодействие между признаками (аналог скрытых слоев в нейросети)
- Измерение = проекция результата в классическое число

**Почему это "квантовое"?** Квантовое запутывание позволяет кубитам исследовать пространство состояний размерностью $2^n$, что дает потенциальное преимущество при обработке сложных зависимостей.

## Гибридный ансамбль (Stacking)

### Идея

Stacking объединяет предсказания разных моделей с помощью мета-модели:

$$p_{final}(x) = f_{meta}(p_{RF}(x), p_{VQC}(x))$$

### Реализация

- **Базовые модели:** RandomForest (ROC=0.5655) и лучшая VQC (ZZ_r2_RA_r3, ROC=0.4544)
- **Мета-модель:** LogisticRegression
- **Мета-признаки:** вероятности от базовых моделей

### Результаты

- **Stacking ROC-AUC: 0.5536**
- **Stacking F1 Score: 0.2794**

### Анализ мета-модели

Stacking Model Coefficients:
RandomForest weight: 1.234
VQC weight: -0.456
Intercept: -0.321


**Интерпретация:** Мета-модель дает больший вес RandomForest и отрицательный вес VQC, что указывает на слабую предсказательную способность квантовой модели.

### Визуализация meta-feature space

График в координатах (RandomForest Prob, VQC Prob) показывает:
- Точки плохо разделяются по классам
- VQC вероятности имеют меньшую дисперсию
- Это объясняет низкий вес VQC в stacking модели

## Почему квантовые модели уступают классическим?

### 1. Низкое соотношение сигнал/шум

Финансовые временные ряды характеризуются крайне низким Signal-to-Noise Ratio (SNR). Взаимная информация между признаками и целевой переменной мала:

$$I(X;Y) \ll 1$$

Это означает, что даже теоретически достижимое качество классификации ограничено. Если информация в данных отсутствует, никакое расширение пространства признаков (даже квантовое) не создаст её.

### 2. Математическая интерпретация квантового отображения

Квантовый feature map выполняет нелинейное вложение данных в гильбертово пространство:

$$x \mapsto |\psi(x)\rangle \in \mathcal{H}, \quad \dim(\mathcal{H}) = 2^4 = 16$$

Классификатор использует ожидание наблюдаемой:

$$\phi(x) = \langle \psi(x)| O |\psi(x)\rangle$$

Разные feature maps создают разные геометрии пространства признаков. Однако **если исходные данные не содержат информации, даже богатое гильбертово пространство не поможет**.

### 3. Корреляция ошибок

Низкая корреляция ошибок (≈0.1-0.2) указывает на то, что модели ошибаются по-разному — это потенциально хорошо для ансамблей, но VQC недостаточно сильны сами по себе.

### 4. Barren plateau (бесплодные плато)

При увеличении глубины квантовой схемы градиенты целевой функции экспоненциально затухают:

$$\text{Var}\left(\frac{\partial L}{\partial \theta}\right) \sim \exp(-n)$$

где $n$ — число кубитов или глубина схемы. Это явление, известное как "бесплодное плато", затрудняет оптимизацию и объясняет нестабильность более глубоких конфигураций.

### 5. Ограниченная выразительность при фиксированной архитектуре

Множество функций, реализуемых VQC с заданной архитектурой, ограничено:

$$\mathcal{F}_{VQC} \subsetneq \mathcal{F}_{all}$$

При 4 кубитах размер гильбертова пространства = 16, но реальная выразительность схемы с 2-3 слоями может быть недостаточной.

## Итоговые выводы

1. **Главный результат:** На данных финансовых временных рядов (AAPL) гибридный квантово-классический ансамбль не превзошёл классические модели машинного обучения. Лучший VQC (ROC-AUC = 0.4649) уступает RandomForest (0.5655) и XGBoost (0.5907).

2. **Причина:** Основное ограничение — не архитектура модели, а **низкое соотношение сигнал/шум** в самих данных. Квантовые модели не могут создать информацию там, где её нет. Корреляция признаков с целевой переменной близка к нулю.

3. **Влияние архитектуры:** Увеличение глубины схемы (reps) ведёт к росту вычислительных затрат без значимого прироста качества ($\partial AUC/\partial reps \approx 0$). Это говорит о неэффективности простого усложнения для шумных данных.

4. **Стабильность:** VQC демонстрируют высокую чувствительность к инициализации (std до 0.0385), что делает их менее надёжными по сравнению с классическими моделями.

5. **Ансамбль:** Stacking (RF + VQC) дал ROC-AUC = 0.5536, что немного хуже RandomForest (0.5655). Анализ коэффициентов мета-модели показал, что VQC получает отрицательный вес из-за слабой предсказательной способности.

6. **Лучшая квантовая модель:** Pauli_r1_RA_r2 (ROC-AUC = 0.4649, F1 = 0.4928) — показывает лучшие результаты среди квантовых моделей, но всё равно уступает классическим.

7. **Теоретическая интерпретация:** Наблюдаемые результаты объясняются комбинацией факторов:
   - Низкое SNR в данных
   - Эффект barren plateau при увеличении глубины
   - Ограниченная выразительность фиксированной архитектуры
   - Сложность оптимизации вариационных схем

8. **Квантовое преимущество:** Работа демонстрирует, что **"квантовое преимущество" носит структурно-зависимый характер** и не проявляется на шумных финансовых рядах. Для его достижения необходимы задачи, где:
   - Данные имеют чёткую нелинейную структуру
   - Соотношение сигнал/шум достаточно высоко
   - Классические модели испытывают трудности из-за размерности или сложности
